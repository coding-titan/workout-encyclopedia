The Torso/Limb Programme

By Sebastian Manieri

The Torso workout consists of the following, which can be done in any order:

  • Horizontal Push (Bench, Incline Bench)
  • Horizontal Pull (Row of some kind)
  • Vertical Pull
  • Vertical Push
  • Abs

The Limb workout consists of the following, which can also be done in any order:

  • Leg pushing motion (Squat, Leg Press)
  • Triceps
  • Leg hinge motion (Deadlift, Romanian Deadlift, One Leg Deadlift)
  • Biceps
  • Calf exercise with stretch emphasis
  • Shoulder with mid focus
